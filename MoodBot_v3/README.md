# MoodBot v3

舊版的scikit-learn跟現有的套件衝突，乾脆整個重寫。

## Features

+ [x] 升級linebot至v3，也將回應訊息改為Flex Message美化

+ [x] `!comment`改名為`!review`

+ [x] 使用scikit-learn v1.5.1重新編譯模型

+ [x] 給`!help`添加Quick Reply，方便用戶快速使用

+ [ ] `!review`用於顯示百分比的`fill`元件已棄用，改用其他方式來顯示進度條
